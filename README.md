# Cypress Mini Framework

Mini Cypress E2E test framework demonstrating:
- Page Object Model
- Custom commands
- Fixtures & data-driven tests
- Auth/session handling
- Edge-case testing

## Applications under test
- https://www.saucedemo.com
- https://the-internet.herokuapp.com

## Tech stack
- Cypress
- JavaScript
- Page Objects
- Fixtures
- Custom Commands

## How to run
```bash
npm install
npx cypress open
